## docker-registry
- Run your own docker registry:

- Follow tutorial here: https://www.digitalocean.com/community/tutorials/how-to-set-up-a-private-docker-registry-on-ubuntu-14-04

- Run registry:

```
	# docker-compose up
```

- Login (using https):

```
	# docker login https://YOURDOMAIN
	Login Succeeded
```
